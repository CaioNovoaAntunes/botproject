Guia Rápido de Configuração

Facebook

  1. Acesse Facebook Developers
  2. Crie um App do tipo "Negócios" ou "Consumidor"
  3. Adicione o produto "Facebook Login" e "Graph API"
  4. Gere um Access Token (Página → Ferramentas → Acesso a tokens)
  5. O Page ID está na sua página do Facebook (Sobre → ID da Página)

Instagram

  1. Você precisa de uma Conta Comercial ou de Criador no Instagram
  2. Conecte sua conta do Instagram a uma Página do Facebook
  3. No Facebook Developers, adicione o produto "Instagram Graph API"
  4. Gere um token com as permissões: instagram_basic, instagram_content_publish, pages_show_list
  5. O User ID é encontrado via GET /me/accounts ou GET /me na Graph API

TikTok

  1. Acesse TikTok Developers
  2. Crie um novo App e ative a "Content Posting API"
  3. Configure o Redirect URL para {{ url_for('auth_tiktok_callback', _external=True) }}
  4. Preencha Client Key e Client Secret acima e clique em "Conectar com TikTok"
  5. O Access Token e Open ID serão preenchidos automaticamente
